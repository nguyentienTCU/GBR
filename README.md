# God Bless Retirement Onboarding Portal

God Bless Retirement (GBR) Onboarding Portal is a full-stack internal platform for managing user onboarding and related workflow operations.

## Project Structure

- `frontend/` - Next.js application for the portal UI
- `backend/` - FastAPI service for APIs, auth, user management, DocuSign integration, and QuickBooks Invoice payment.

## Setup Guides

Use the module-specific READMEs for setup and local development:

- Backend setup: [backend/README.md](./backend/README.md)
- Frontend setup: [frontend/README.md](./frontend/README.md)

## Stack

- Frontend: Next.js
- Backend: FastAPI
- Auth/Data: Supabase
- E-signature: DocuSign
- Payment: QuickBooks Invoice
